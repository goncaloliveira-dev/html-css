Codigos e Pré-Formatações

Usar <code> para o texto ficar com formato de código: document.getElementbyID('teste')

Mas usar <pre> dentro do <code> </pre> para fazer quebra de linha como num código real:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print ('Número par')
    else:
        print ('Número impar')
